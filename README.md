# AUV Terrain Localisation

ROS 2 based terrain-relative navigation framework for GPS-denied AUV localisation.

## Objectives

- Global localisation with unknown initial pose
- Terrain-aided Monte Carlo localisation
- Bathymetric map matching
- Real-time ROS 2 implementation
- Under-ice navigation simulation

## Software Stack

- ROS 2 Jazzy
- Gazebo Harmonic
- Docker
- C++
- Python
